Responsive Web Design

¿Qué es?

Es una técnica de diseño y desarrollo web que mediante el uso de estructuras flexibles (contenedores flexibles, imágenes y video flexibles) y junto con Media Queries especificados en CSS, logran adaptar un sitio web al entorno del dispositivo en el que se encuentre.

Con ésta técnica se consigue que el contenido del diseño de tu sitio se vea bien y pueda ser bastante legible para los usuarios sobre el dispositivo con el que éste interactúe, sea un celular, tablet, pc de escritorio o en un televisor.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

¿Qué es?

En español “consulta de medios” nos permite consultar e identificar el medio en el cuál se está mostrando nuestro sitio. Si la consulta resulta verdadera entonces se aplicarán determinados estilos CSS para este tipo de medio

Bootstrap

¿Qué es?

Bootstrap es una librería de clases de código abierto para facilitar el armado front-end de un sitio.

¿Para qué sirve?

Principalmente para ahorrar tiempo en el diseño del sitio, usando estructuras y clases pre-formateadas.

Ventajas de usar Bootstrap


  • Mantenimiento y actualización:
    No es una herramienta sea perfecta, pero gran parte del trabajo interno ya está llevado a cabo por sus creadores (Twitter).

  • Paquete de elementos web customizables:
    Se puede diseñar una web jugando con sus elementos y estos siempre encajan.

  • Componentes vitales para los desarrolladores:
    Como HTML5, CSS3, jQuery o GitHub, entre otros.

  • Plantillas responsive:
    Se desarrolló con la idea de facilitar el proceso de adaptación web a todo tipo de dispositivos.

  • Incluye Grid system:
    Muy útil para maquetar por columnas.

Ofrece un paquete de elementos web customizables: Con Bootstrap puedes diseñar una web jugando con sus elementos compuestos por diferentes combinaciones de HTML, CSS y Javascript, de manera que las piezas siempre encajan.

-Ventaja #3

Utiliza componentes vitales para los desarrolladores: Como HTML5, CSS3, jQuery o GitHub, entre otros.

Tablas

Para el manejo de tablas, bootstrap maneja diversas clases, que permiten mejorar la funcionalidad y estética de las tablas. Algunos ejemplos de dichas clases son:

  • table table-striped : Permite alternar colores de fondo en las filas
  • table table-bordered: Permite establecer bordes horizontales y verticales
  • table table-hover: Al pasar el mouse por las filas cambia el aspecto de la fila actual
  • table table-dark table-striped: Anidamiento de dos clases, le da una tonalidad oscura y alternada a las filas
  • table table-dark table-hover: Establece tonalidad oscuras alternadas en las filas, cuando se hace foco con el mouse
  • table table-primary: Establece una estética para las filas
  • table table-responsive-sm: Permite que la tabla se vuelva responsiva cuando la pantalla es sm(small)
